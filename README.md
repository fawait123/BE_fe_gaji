# PROJECT

project with react library.

## Installation

install modules react

```bash
# with yarn
yarn

# with npm
npm install
```

## Usage

```python
# with yarn
yarn start

# with npm 
npm start

# build with yarn 
yarn build

# build with npm
npm run build
```
